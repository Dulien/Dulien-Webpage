Handwritten Digits OCR

Using Tensorflow and Keras libraries, I created a basic neural network that could correctly identify any handwritten digits (from 0 - 9). The model uses the MNIST handwrittem digits database to train itself. The dataset consists of 60,000 training images and 10,000 testing images. The images are in a gray-scale value and they each have a dimension of 28x28 pixels. Alongside my friend Stefano Stella, we created a basic user interface for this program using PyQt5.
